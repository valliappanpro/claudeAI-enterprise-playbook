## 📌 About

Claude AI enterprise playbook: prompt engineering patterns, Claude Projects design, LLM document automation, AI program ROI measurement, and adoption frameworks. Field-tested across healthcare IT consulting, care management, and payer operations. Open-source templates for AI leads and consultants.

This playbook is distilled from leading AI programs across 200+ person professional services organizations in the healthcare payer domain — spanning care management, utilization management, claims operations, and enterprise implementations. It is designed for **AI leads, solution architects, consultants, and implementation teams** looking to move beyond prototypes into production-scale AI programs.

## 🎯 What's Inside

### 1. Prompt Engineering Patterns
Battle-tested prompt architectures for enterprise use cases — instruction design, system prompt structuring, few-shot templates, and chain-of-thought strategies tailored for domain-heavy consulting workflows.

### 2. Claude Projects Architecture
Design patterns for building Claude Projects that serve as domain-specific AI assistants — from business rules engines to regulatory compliance tools. Includes instruction set templates and knowledge base organization strategies.

### 3. Document Automation Pipelines
End-to-end patterns for generating professional deliverables at scale — PowerPoint decks, Word reports, PDF documents, and Excel trackers using Claude with libraries like pptxgenjs, python-docx, ReportLab, and openpyxl.

### 4. AI Adoption Measurement
Frameworks for measuring AI program impact using Lean Six Sigma methodology — survey design, usage analytics, ROI models, and executive reporting templates. Includes a proven adoption strategy targeting measurable savings and throughput improvements.

### 5. Enterprise Integration
Patterns for connecting Claude with enterprise tools — Jira analytics, Confluence automation, Smartsheet workflows, and API orchestration with platforms like AWS Bedrock.

### 6. Healthcare AI Patterns
Industry-specific patterns for healthcare payer operations — care management, utilization management, claims processing, and regulatory compliance. Includes considerations for HL7/FHIR data standards and HIPAA.

### 7. Resources
Curated tools, reading lists, glossary, and a changelog tracking playbook evolution.

## 🚀 Quick Start

1. **Browse by topic** — Each folder has its own `README.md` with context and navigation.
2. **Start with fundamentals** — If you're new to Claude in enterprise settings, begin with `01-prompt-engineering/` and `02-claude-projects/`.
3. **Grab templates** — Check the `templates/` and `examples/` subfolders for reusable artifacts.
4. **Measure impact** — Use `04-ai-adoption-measurement/` to build your own AI ROI framework.
